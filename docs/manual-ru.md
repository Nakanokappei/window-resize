## Первоначальная настройка

### Предоставление разрешения Универсального доступа

Window Resize использует API Универсального доступа macOS для изменения размера окон. При первом запуске приложения необходимо предоставить соответствующее разрешение.

1. Запустите **Window Resize**. Появится системный диалог с запросом на предоставление доступа к Универсальному доступу.
2. Нажмите **«Открыть настройки»** (или перейдите вручную в **Системные настройки > Конфиденциальность и безопасность > Универсальный доступ**).
3. Найдите **«Window Resize»** в списке и включите переключатель.
4. Вернитесь к приложению — в строке меню появится иконка, и приложение будет готово к использованию.

> **Примечание:** Если диалог не появился, вы можете открыть настройки Универсального доступа непосредственно из окна настроек приложения (см. [Состояние Универсального доступа](#состояние-универсального-доступа)).

## Изменение размера окна

### Пошаговая инструкция

1. Нажмите на **иконку Window Resize** в строке меню.
2. Наведите курсор на **«Изменить размер»**, чтобы открыть список окон.
3. Все открытые окна отображаются с **иконкой приложения** и именем в формате **[Название приложения] Заголовок окна**. Длинные заголовки автоматически сокращаются для удобства чтения меню.
4. Если у приложения **3 или более окон**, они автоматически группируются под названием приложения (например, **«Safari (4)»**). Наведите курсор на приложение, чтобы увидеть отдельные окна, а затем на окно — чтобы увидеть доступные размеры.
5. Наведите курсор на нужное окно, чтобы увидеть доступные предустановленные размеры.
6. Нажмите на размер, чтобы мгновенно изменить размер окна.

### Отображение размеров

Каждый элемент размера в меню отображается следующим образом:

```
1920 x 1080          Full HD
```

- **Слева:** Ширина x Высота (в пикселях)
- **Справа:** Название (имя устройства или стандарта), отображается серым цветом

### Размеры, превышающие размер экрана

Если предустановленный размер превышает разрешение дисплея, на котором расположено окно, этот размер будет **выделен серым и недоступен для выбора**. Это предотвращает изменение размера окна за пределы границ экрана.

> **Несколько дисплеев:** Приложение определяет, на каком дисплее находится каждое окно, и соответственно корректирует доступные размеры.

## Настройки

Откройте настройки из строки меню: нажмите на иконку Window Resize, затем выберите **«Настройки...»** (горячая клавиша: **⌘,**).

### Встроенные размеры

Приложение содержит 12 встроенных предустановленных размеров:

| Размер | Название |
|--------|----------|
| 2560 x 1600 | MacBook Pro 16" |
| 2560 x 1440 | QHD / iMac |
| 1728 x 1117 | MacBook Pro 14" |
| 1512 x 982 | MacBook Air 15" |
| 1470 x 956 | MacBook Air 13" M3 |
| 1440 x 900 | MacBook Air 13" |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1280 x 800 | WXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

Встроенные размеры нельзя удалить или отредактировать.

### Пользовательские размеры

Вы можете добавить собственные размеры в список:

1. В разделе **«Пользовательские»** введите **Ширину** и **Высоту** в пикселях.
2. Нажмите **«Добавить»**.
3. Новый размер появится в пользовательском списке и сразу станет доступен в меню изменения размера.

Чтобы удалить пользовательский размер, нажмите красную кнопку **«Удалить»** рядом с ним.

> Пользовательские размеры отображаются в меню после встроенных размеров.

### Запускать при входе

Включите переключатель **«Запускать при входе»**, чтобы Window Resize автоматически запускалось при входе в macOS.

## Специальные возможности

Перечисленные ниже функции расширяют возможности управления окнами. При включении любой из них в меню размеров появляется пункт **«Текущий размер»**, позволяющий переместить окно или вынести его на передний план без изменения размера.

### Вынести на передний план

Включите переключатель **«Выносить окно на передний план после изменения размера»**, чтобы автоматически поднимать окно поверх всех остальных. Это удобно, когда нужное окно частично скрыто за другими.

### Переместить на основной экран

Включите переключатель **«Переместить на основной экран»**, чтобы при изменении размера окно автоматически перемещалось на главный дисплей. Это полезно при работе с несколькими мониторами, когда нужно быстро перенести окно с дополнительного экрана.

### Расположение окна

Выберите, куда поместить окно на экране после изменения размера. Ряд из 9 кнопок представляет варианты расположения:

- **Углы:** Верхний левый, Верхний правый, Нижний левый, Нижний правый
- **Края:** По центру сверху, По центру слева, По центру справа, По центру снизу
- **Центр:** В центре экрана

Нажмите на кнопку расположения, чтобы выбрать его. Нажмите ту же кнопку повторно (или нажмите **«Не перемещать»**), чтобы снять выбор. Если расположение не выбрано, окно останется на прежнем месте после изменения размера.

> **Примечание:** При расположении окна учитываются строка меню и Dock, поэтому окно размещается в пределах рабочей области экрана.

---

### Снимок экрана

Включите переключатель **«Сделать снимок после изменения размера»**, чтобы автоматически делать снимок окна после изменения его размера.

Когда функция включена, доступны следующие параметры:

- **Сохранить в файл** — Сохранить снимок экрана в файл PNG. Нажмите **«Выбрать...»**, чтобы указать папку для сохранения.
  > **Формат имени файла:** `MMddHHmmss_ИмяПриложения_ЗаголовокОкна.png` (например, `0227193012_Safari_Apple.png`). Символы удаляются; используются только буквы, цифры и символы подчёркивания.
- **Скопировать в буфер обмена** — Скопировать снимок экрана в буфер обмена для вставки в другие приложения.

Оба варианта можно включить независимо друг от друга. Например, можно скопировать в буфер обмена, не сохраняя в файл.

> **Примечание:** Функция снимка экрана требует разрешения **Запись экрана**. При первом использовании macOS предложит предоставить разрешение в **Системные настройки > Конфиденциальность и безопасность > Запись экрана**.

### Язык

Выберите язык интерфейса приложения в раскрывающемся списке **«Язык»**. Доступно 16 языков, а также вариант **«Системный язык»**, при котором приложение следует языку macOS. Для применения нового языка требуется перезапуск приложения.

### Состояние Универсального доступа

В нижней части окна настроек индикатор состояния показывает текущее состояние разрешения Универсального доступа:

| Индикатор | Значение |
|-----------|----------|
| 🟢 **Универсальный доступ: Включён** | Разрешение активно и работает корректно. |
| 🟠 **Универсальный доступ: Требуется обновление** | Система сообщает, что разрешение предоставлено, но оно больше недействительно (см. [Исправление устаревших разрешений](#исправление-устаревших-разрешений)). Отображается кнопка **«Открыть настройки»**. |
| 🔴 **Универсальный доступ: Отключён** | Разрешение не предоставлено. Отображается кнопка **«Открыть настройки»**. |

## Устранение неполадок

### Исправление устаревших разрешений

Если вы видите оранжевый индикатор состояния или сообщение «Универсальный доступ: Требуется обновление», разрешение стало устаревшим. Это может произойти после обновления или пересборки приложения.

**Для исправления:**

1. Откройте **Системные настройки > Конфиденциальность и безопасность > Универсальный доступ**.
2. Найдите **«Window Resize»** в списке.
3. Переключите его в положение **ВЫКЛ**, затем снова в положение **ВКЛ**.
4. Или же удалите его из списка полностью, а затем перезапустите приложение для повторного добавления.

### Не удалось изменить размер

Если появляется предупреждение «Не удалось изменить размер», возможные причины:

- Целевое приложение не поддерживает изменение размера через Универсальный доступ.
- Окно находится в **полноэкранном режиме** (сначала выйдите из полноэкранного режима).
- Разрешение Универсального доступа не активно (проверьте состояние в Настройках).

### Окно не отображается в списке

Меню изменения размера отображает только те окна, которые:

- В данный момент видимы на экране
- Не являются частью рабочего стола (например, рабочий стол Finder исключён)
- Не являются собственными окнами приложения Window Resize

Если окно свёрнуто в Dock, оно не будет отображаться в списке.

### Снимок экрана не работает

Если снимки экрана не создаются:

- Предоставьте разрешение **Запись экрана** в **Системные настройки > Конфиденциальность и безопасность > Запись экрана**.
- Убедитесь, что включён хотя бы один из вариантов: **«Сохранить в файл»** или **«Скопировать в буфер обмена»**.
