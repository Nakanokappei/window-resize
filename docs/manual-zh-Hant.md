## 初始設定

### 授予輔助使用權限

Window Resize 使用 macOS 輔助使用 API 來調整視窗大小。首次啟動時需要授予權限。

1. 啟動 **Window Resize**。系統會彈出對話框，要求您授予輔助使用存取權限。
2. 點擊 **「打開設定」**（或手動前往 **系統設定 > 隱私權與安全性 > 輔助使用**）。
3. 在列表中找到 **「Window Resize」**，打開開關。
4. 返回應用程式——選單列圖示將出現，應用程式即可使用。

> **注意：** 如果對話框未出現，您可以從應用程式的設定視窗直接打開輔助使用設定（請參閱[輔助使用狀態](#輔助使用狀態)）。

## 調整視窗大小

### 操作步驟

1. 點擊選單列中的 **Window Resize 圖示**。
2. 將滑鼠懸停在 **「調整大小」** 上，打開視窗列表。
3. 所有目前開啟的視窗將以 **[應用程式名稱] 視窗標題** 的格式列出。
4. 將滑鼠懸停在某個視窗上，查看可用的預設大小。
5. 點擊某個大小，即可立即調整視窗。

### 大小的顯示方式

選單中的每個大小項目顯示如下：

```
1920 x 1080          Full HD
```

- **左側：** 寬 x 高（以像素為單位）
- **右側：** 標籤（裝置名稱或標準名稱），以灰色顯示

### 超出螢幕的大小

如果預設大小超過視窗所在顯示器的解析度，該大小將 **顯示為灰色且無法選取**。這可以防止將視窗調整到超出螢幕邊界。

> **多顯示器：** 應用程式會偵測每個視窗所在的顯示器，並據此調整可用的大小選項。

## 設定

從選單列打開設定：點擊 Window Resize 圖示，然後選擇 **「設定⋯」**（快速鍵：**⌘,**）。

### 內建大小

應用程式包含 12 個內建預設大小：

| 大小 | 標籤 |
|------|------|
| 2560 x 1600 | MacBook Pro 16" |
| 2560 x 1440 | QHD / iMac |
| 1728 x 1117 | MacBook Pro 14" |
| 1512 x 982 | MacBook Air 15" |
| 1470 x 956 | MacBook Air 13" M3 |
| 1440 x 900 | MacBook Air 13" |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1280 x 800 | WXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

內建大小無法移除或編輯。

### 自訂大小

您可以向列表中加入自己的大小：

1. 在 **「自訂」** 部分，輸入 **寬度** 和 **高度**（以像素為單位）。
2. 點擊 **「加入」**。
3. 新大小將出現在自訂列表中，並立即可在調整大小選單中使用。

要移除自訂大小，請點擊旁邊紅色的 **「移除」** 按鈕。

> 自訂大小在調整大小選單中顯示在內建大小之後。

### 登入時啟動

打開 **「登入時啟動」** 開關，可讓 Window Resize 在您登入 macOS 時自動啟動。

### 截圖

打開 **「調整大小後截圖」** 開關，可在調整視窗大小後自動擷取視窗圖片。

啟用後，以下選項可用：

- **儲存到檔案** — 將截圖儲存為 PNG 檔案。啟用後，選擇儲存位置：
  - **桌面** — 儲存到桌面資料夾。
  - **圖片** — 儲存到圖片資料夾。
- **拷貝到剪貼板** — 將截圖拷貝到剪貼板，方便貼上到其他應用程式。

兩個選項可以獨立啟用。例如，您可以僅拷貝到剪貼板而不儲存檔案。

> **注意：** 截圖功能需要 **螢幕錄製** 權限。首次使用此功能時，macOS 會提示您前往 **系統設定 > 隱私權與安全性 > 螢幕錄製** 授予權限。

### 輔助使用狀態

在設定視窗底部，狀態指示器顯示輔助使用權限的目前狀態：

| 指示器 | 含義 |
|--------|------|
| 🟢 **輔助使用：已啟用** | 權限已啟用且運作正常。 |
| 🟠 **輔助使用：需要重新整理** | 系統報告權限已授予，但已失效（請參閱[修復失效權限](#修復失效權限)）。會顯示 **「打開設定」** 按鈕。 |
| 🔴 **輔助使用：未啟用** | 權限尚未授予。會顯示 **「打開設定」** 按鈕。 |

## 疑難排解

### 修復失效權限

如果您看到橘色狀態指示器或「輔助使用：需要重新整理」的提示，表示權限已失效。這可能在應用程式更新或重新建置後發生。

**修復方法：**

1. 打開 **系統設定 > 隱私權與安全性 > 輔助使用**。
2. 在列表中找到 **「Window Resize」**。
3. 將其 **關閉**，然後重新 **開啟**。
4. 或者，將其從列表中完全移除，然後重新啟動應用程式以重新加入。

### 調整大小失敗

如果您看到「調整大小失敗」的提示，可能的原因包括：

- 目標應用程式不支援透過輔助使用調整大小。
- 視窗處於 **全螢幕模式**（請先退出全螢幕）。
- 輔助使用權限未啟用（請在設定中檢查狀態）。

### 視窗未出現在列表中

調整大小選單僅顯示以下視窗：

- 目前在螢幕上可見的視窗
- 不屬於桌面的視窗（例如 Finder 桌面會被排除）
- 不是 Window Resize 應用程式自身的視窗

如果視窗已最小化到 Dock，則不會出現在列表中。

### 截圖不起作用

如果截圖未被擷取：

- 請在 **系統設定 > 隱私權與安全性 > 螢幕錄製** 中授予 **螢幕錄製** 權限。
- 確保 **「儲存到檔案」** 或 **「拷貝到剪貼板」** 中至少有一個已啟用。
