## การตั้งค่าเริ่มต้น

### การให้สิทธิ์การช่วยการเข้าถึง

Window Resize ใช้ API การช่วยการเข้าถึงของ macOS เพื่อปรับขนาดหน้าต่าง คุณต้องให้สิทธิ์เมื่อเปิดใช้งานแอปครั้งแรก

1. เปิด **Window Resize** จะปรากฏกล่องโต้ตอบของระบบขอให้คุณอนุญาตการเข้าถึงการช่วยการเข้าถึง
2. คลิก **"เปิดการตั้งค่า"** (หรือไปที่ **การตั้งค่าระบบ > ความเป็นส่วนตัวและความปลอดภัย > การช่วยการเข้าถึง** ด้วยตนเอง)
3. ค้นหา **"Window Resize"** ในรายการและเปิดสวิตช์
4. กลับไปที่แอป — ไอคอนแถบเมนูจะปรากฏขึ้นและแอปพร้อมใช้งาน

> **หมายเหตุ:** หากกล่องโต้ตอบไม่ปรากฏ คุณสามารถเปิดการตั้งค่าการช่วยการเข้าถึงได้โดยตรงจากหน้าต่างการตั้งค่าของแอป (ดู [สถานะการช่วยการเข้าถึง](#สถานะการช่วยการเข้าถึง))

## การปรับขนาดหน้าต่าง

### ขั้นตอนทีละขั้น

1. คลิก **ไอคอน Window Resize** บนแถบเมนู
2. ชี้เมาส์ไปที่ **"ปรับขนาด"** เพื่อเปิดรายการหน้าต่าง
3. หน้าต่างทั้งหมดที่เปิดอยู่จะแสดงพร้อม**ไอคอนแอปพลิเคชัน**และชื่อเป็น **[ชื่อแอป] ชื่อหน้าต่าง** ชื่อที่ยาวจะถูกตัดให้สั้นลงโดยอัตโนมัติเพื่อให้เมนูอ่านง่าย
4. เมื่อแอปพลิเคชันมี **หน้าต่างตั้งแต่ 3 หน้าต่างขึ้นไป** หน้าต่างเหล่านั้นจะถูกจัดกลุ่มไว้ใต้ชื่อแอปโดยอัตโนมัติ (เช่น **"Safari (4)"**) ชี้เมาส์ไปที่ชื่อแอปเพื่อดูหน้าต่างแต่ละรายการ จากนั้นชี้ไปที่หน้าต่างเพื่อดูขนาดที่ใช้ได้
5. ชี้เมาส์ไปที่หน้าต่างเพื่อดูขนาดที่ตั้งไว้ที่ใช้ได้
6. คลิกที่ขนาดเพื่อปรับขนาดหน้าต่างทันที

### วิธีการแสดงขนาด

แต่ละรายการขนาดในเมนูจะแสดงดังนี้:

```
1920 x 1080          Full HD
```

- **ด้านซ้าย:** ความกว้าง x ความสูง (หน่วยพิกเซล)
- **ด้านขวา:** ป้ายชื่อ (ชื่ออุปกรณ์หรือชื่อมาตรฐาน) แสดงเป็นสีเทา

### ขนาดที่เกินขนาดจอภาพ

หากขนาดที่ตั้งไว้มีขนาดใหญ่กว่าจอภาพที่หน้าต่างอยู่ ขนาดนั้นจะ **เป็นสีเทาและไม่สามารถเลือกได้** เพื่อป้องกันไม่ให้คุณปรับขนาดหน้าต่างเกินขอบเขตของจอภาพ

> **หลายจอภาพ:** แอปจะตรวจจับว่าแต่ละหน้าต่างอยู่บนจอภาพใดและปรับขนาดที่ใช้ได้โดยอัตโนมัติ

## การตั้งค่า

เปิดการตั้งค่าจากแถบเมนู: คลิกไอคอน Window Resize จากนั้นเลือก **"การตั้งค่า..."** (ทางลัด: **⌘,**)

### ขนาดในตัว

แอปมีขนาดที่ตั้งไว้ในตัว 12 ขนาด:

| ขนาด | ป้ายชื่อ |
|------|----------|
| 2560 x 1600 | MacBook Pro 16" |
| 2560 x 1440 | QHD / iMac |
| 1728 x 1117 | MacBook Pro 14" |
| 1512 x 982 | MacBook Air 15" |
| 1470 x 956 | MacBook Air 13" M3 |
| 1440 x 900 | MacBook Air 13" |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1280 x 800 | WXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

ขนาดในตัวไม่สามารถลบหรือแก้ไขได้

### ขนาดกำหนดเอง

คุณสามารถเพิ่มขนาดของคุณเองลงในรายการ:

1. ในส่วน **"กำหนดเอง"** ให้ป้อน **ความกว้าง** และ **ความสูง** เป็นพิกเซล
2. คลิก **"เพิ่ม"**
3. ขนาดใหม่จะปรากฏในรายการกำหนดเองและสามารถใช้ได้ทันทีในเมนูปรับขนาด

หากต้องการลบขนาดกำหนดเอง ให้คลิกปุ่ม **"ลบ"** สีแดงที่อยู่ถัดจากรายการนั้น

> ขนาดกำหนดเองจะแสดงในเมนูปรับขนาดหลังจากขนาดในตัว

### เปิดเมื่อเข้าสู่ระบบ

เปิด **"เปิดเมื่อเข้าสู่ระบบ"** เพื่อให้ Window Resize เริ่มทำงานโดยอัตโนมัติเมื่อคุณเข้าสู่ระบบ macOS

## คุณสมบัติการช่วยการเข้าถึง

คุณสมบัติต่อไปนี้ช่วยเพิ่มความสะดวกในการจัดการหน้าต่าง เมื่อเปิดใช้งานคุณสมบัติใดคุณสมบัติหนึ่ง เมนูปรับขนาดจะแสดงตัวเลือก **"ขนาดปัจจุบัน"** ไว้ด้านบน ช่วยให้คุณจัดตำแหน่งหรือนำหน้าต่างมาด้านหน้าได้โดยไม่ต้องเปลี่ยนขนาด

### นำมาด้านหน้า

เปิด **"นำหน้าต่างมาด้านหน้าหลังปรับขนาด"** เพื่อยกหน้าต่างที่ปรับขนาดแล้วขึ้นมาอยู่เหนือหน้าต่างอื่นทั้งหมดโดยอัตโนมัติ มีประโยชน์เมื่อหน้าต่างเป้าหมายถูกบังอยู่ด้านหลังหน้าต่างอื่น

### ย้ายไปจอภาพหลัก

เปิด **"ย้ายไปจอภาพหลัก"** เพื่อย้ายหน้าต่างไปยังจอภาพหลักเมื่อปรับขนาด มีประโยชน์ในการใช้งานหลายจอภาพเมื่อคุณต้องการย้ายหน้าต่างจากจอภาพรองอย่างรวดเร็ว

### ตำแหน่งหน้าต่าง

เลือกตำแหน่งที่จะวางหน้าต่างบนจอภาพหลังจากปรับขนาด ปุ่ม 9 ปุ่มแสดงตัวเลือกตำแหน่ง:

- **มุมทั้งสี่:** บนซ้าย, บนขวา, ล่างซ้าย, ล่างขวา
- **กึ่งกลางด้านข้าง:** กลางด้านบน, กลางด้านซ้าย, กลางด้านขวา, กลางด้านล่าง
- **กึ่งกลาง:** กลางจอภาพ

คลิกปุ่มตำแหน่งเพื่อเลือก คลิกปุ่มเดิมอีกครั้ง (หรือคลิก **"ไม่ย้าย"**) เพื่อยกเลิกการเลือก เมื่อไม่ได้เลือกตำแหน่งใด หน้าต่างจะอยู่ที่เดิมหลังจากปรับขนาด

> **หมายเหตุ:** การจัดตำแหน่งหน้าต่างคำนึงถึงแถบเมนูและ Dock โดยวางหน้าต่างไว้ในพื้นที่ใช้งานได้ของจอภาพ

---

### ภาพหน้าจอ

เปิด **"ถ่ายภาพหน้าจอหลังปรับขนาด"** เพื่อจับภาพหน้าต่างโดยอัตโนมัติหลังจากปรับขนาด

เมื่อเปิดใช้งานแล้ว ตัวเลือกต่อไปนี้จะพร้อมใช้งาน:

- **บันทึกเป็นไฟล์** — บันทึกภาพหน้าจอเป็นไฟล์ PNG คลิก **"เลือก..."** เพื่อเลือกโฟลเดอร์สำหรับบันทึก
  > **รูปแบบชื่อไฟล์:** `MMddHHmmss_ชื่อแอป_ชื่อหน้าต่าง.png` (เช่น `0227193012_Safari_Apple.png`) สัญลักษณ์จะถูกลบออก ใช้เฉพาะตัวอักษร ตัวเลข และขีดล่างเท่านั้น
- **คัดลอกไปยังคลิปบอร์ด** — คัดลอกภาพหน้าจอไปยังคลิปบอร์ดเพื่อวางในแอปอื่น

ทั้งสองตัวเลือกสามารถเปิดใช้งานแยกกันได้ ตัวอย่างเช่น คุณสามารถคัดลอกไปยังคลิปบอร์ดโดยไม่ต้องบันทึกเป็นไฟล์

> **หมายเหตุ:** ฟีเจอร์ภาพหน้าจอต้องมีสิทธิ์ **บันทึกหน้าจอ** เมื่อคุณใช้ฟีเจอร์นี้ครั้งแรก macOS จะแจ้งให้คุณอนุญาตสิทธิ์ใน **การตั้งค่าระบบ > ความเป็นส่วนตัวและความปลอดภัย > บันทึกหน้าจอ**

### ภาษา

เลือกภาษาที่แสดงในแอปจากเมนูดรอปดาวน์ **ภาษา** เลือกจาก 16 ภาษา หรือเลือก **"ค่าเริ่มต้นของระบบ"** เพื่อใช้ภาษาตามการตั้งค่าระบบ macOS การเปลี่ยนภาษาจะมีผลหลังจากเปิดแอปใหม่

### สถานะการช่วยการเข้าถึง

ที่ด้านล่างของหน้าต่างการตั้งค่า ตัวบ่งชี้สถานะจะแสดงสถานะปัจจุบันของสิทธิ์การช่วยการเข้าถึง:

| ตัวบ่งชี้ | ความหมาย |
|-----------|----------|
| 🟢 **การช่วยการเข้าถึง: เปิดใช้งานแล้ว** | สิทธิ์ใช้งานได้และทำงานอย่างถูกต้อง |
| 🟠 **การช่วยการเข้าถึง: ต้องรีเฟรช** | ระบบรายงานว่าได้รับสิทธิ์แล้ว แต่ไม่สามารถใช้งานได้อีกต่อไป (ดู [การแก้ไขสิทธิ์ที่หมดอายุ](#การแก้ไขสิทธิ์ที่หมดอายุ)) ปุ่ม **"เปิดการตั้งค่า"** จะแสดงขึ้น |
| 🔴 **การช่วยการเข้าถึง: ไม่ได้เปิดใช้งาน** | ยังไม่ได้ให้สิทธิ์ ปุ่ม **"เปิดการตั้งค่า"** จะแสดงขึ้น |

## การแก้ไขปัญหา

### การแก้ไขสิทธิ์ที่หมดอายุ

หากคุณเห็นตัวบ่งชี้สถานะสีส้มหรือข้อความ "การช่วยการเข้าถึง: ต้องรีเฟรช" แสดงว่าสิทธิ์หมดอายุแล้ว สิ่งนี้อาจเกิดขึ้นหลังจากแอปได้รับการอัปเดตหรือถูกบิลด์ใหม่

**วิธีแก้ไข:**

1. เปิด **การตั้งค่าระบบ > ความเป็นส่วนตัวและความปลอดภัย > การช่วยการเข้าถึง**
2. ค้นหา **"Window Resize"** ในรายการ
3. **ปิด** สวิตช์ จากนั้น **เปิด** อีกครั้ง
4. หรือลบออกจากรายการทั้งหมด จากนั้นเปิดแอปอีกครั้งเพื่อเพิ่มกลับ

### ปรับขนาดไม่สำเร็จ

หากคุณเห็นการแจ้งเตือน "ปรับขนาดไม่สำเร็จ" สาเหตุที่เป็นไปได้ ได้แก่:

- แอปพลิเคชันเป้าหมายไม่รองรับการปรับขนาดผ่านการช่วยการเข้าถึง
- หน้าต่างอยู่ใน **โหมดเต็มหน้าจอ** (ออกจากโหมดเต็มหน้าจอก่อน)
- สิทธิ์การช่วยการเข้าถึงไม่ได้เปิดใช้งาน (ตรวจสอบสถานะในการตั้งค่า)

### หน้าต่างไม่แสดงในรายการ

เมนูปรับขนาดจะแสดงเฉพาะหน้าต่างที่:

- แสดงอยู่บนหน้าจอในขณะนั้น
- ไม่ใช่ส่วนหนึ่งของเดสก์ท็อป (เช่น เดสก์ท็อป Finder จะถูกยกเว้น)
- ไม่ใช่หน้าต่างของแอป Window Resize เอง

หากหน้าต่างถูกย่อลงไปที่ Dock จะไม่ปรากฏในรายการ

### ภาพหน้าจอไม่ทำงาน

หากภาพหน้าจอไม่ถูกจับภาพ:

- ให้สิทธิ์ **บันทึกหน้าจอ** ใน **การตั้งค่าระบบ > ความเป็นส่วนตัวและความปลอดภัย > บันทึกหน้าจอ**
- ตรวจสอบว่าอย่างน้อยหนึ่งตัวเลือกจาก **"บันทึกเป็นไฟล์"** หรือ **"คัดลอกไปยังคลิปบอร์ด"** ถูกเปิดใช้งานอยู่
