## 初始设置

### 授予辅助功能权限

Window Resize 使用 macOS 辅助功能 API 来调整窗口大小。首次启动时需要授予权限。

1. 启动 **Window Resize**。系统会弹出对话框，要求您授予辅助功能访问权限。
2. 点击 **"打开系统设置"**（或手动前往 **系统设置 > 隐私与安全性 > 辅助功能**）。
3. 在列表中找到 **"Window Resize"**，打开开关。
4. 返回应用——菜单栏图标将出现，应用即可使用。

> **注意：** 如果对话框未出现，您可以从应用的设置窗口直接打开辅助功能设置（请参阅[辅助功能状态](#辅助功能状态)）。

## 调整窗口大小

### 操作步骤

1. 点击菜单栏中的 **Window Resize 图标**。
2. 将鼠标悬停在 **"调整大小"** 上，打开窗口列表。
3. 所有当前打开的窗口将以 **[应用名称] 窗口标题** 的格式列出。
4. 将鼠标悬停在某个窗口上，查看可用的预设大小。
5. 点击某个大小，即可立即调整窗口。

### 大小的显示方式

菜单中的每个大小条目显示如下：

```
1920 x 1080          Full HD
```

- **左侧：** 宽 x 高（以像素为单位）
- **右侧：** 标签（设备名称或标准名称），以灰色显示

### 超出屏幕的大小

如果预设大小超过窗口所在显示器的分辨率，该大小将 **显示为灰色且无法选择**。这可以防止将窗口调整到超出屏幕边界。

> **多显示器：** 应用会检测每个窗口所在的显示器，并据此调整可用的大小选项。

## 设置

从菜单栏打开设置：点击 Window Resize 图标，然后选择 **"设置…"**（快捷键：**⌘,**）。

### 内置大小

应用包含 12 个内置预设大小：

| 大小 | 标签 |
|------|------|
| 2560 x 1600 | MacBook Pro 16" |
| 2560 x 1440 | QHD / iMac |
| 1728 x 1117 | MacBook Pro 14" |
| 1512 x 982 | MacBook Air 15" |
| 1470 x 956 | MacBook Air 13" M3 |
| 1440 x 900 | MacBook Air 13" |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1280 x 800 | WXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

内置大小无法移除或编辑。

### 自定义大小

您可以向列表中添加自己的大小：

1. 在 **"自定义"** 部分，输入 **宽度** 和 **高度**（以像素为单位）。
2. 点击 **"添加"**。
3. 新大小将出现在自定义列表中，并立即可在调整大小菜单中使用。

要移除自定义大小，请点击旁边红色的 **"移除"** 按钮。

> 自定义大小在调整大小菜单中显示在内置大小之后。

### 登录时启动

打开 **"登录时启动"** 开关，可让 Window Resize 在您登录 macOS 时自动启动。

### 截图

打开 **"调整大小后截图"** 开关，可在调整窗口大小后自动截取窗口图片。

启用后，以下选项可用：

- **保存到文件** — 将截图保存为 PNG 文件。启用后，选择保存位置：
  - **桌面** — 保存到桌面文件夹。
  - **图片** — 保存到图片文件夹。
- **复制到剪贴板** — 将截图复制到剪贴板，方便粘贴到其他应用。

两个选项可以独立启用。例如，您可以仅复制到剪贴板而不保存文件。

> **注意：** 截图功能需要 **屏幕录制** 权限。首次使用此功能时，macOS 会提示您前往 **系统设置 > 隐私与安全性 > 屏幕录制** 授予权限。

### 辅助功能状态

在设置窗口底部，状态指示器显示辅助功能权限的当前状态：

| 指示器 | 含义 |
|--------|------|
| 🟢 **辅助功能：已启用** | 权限已激活且工作正常。 |
| 🟠 **辅助功能：需要刷新** | 系统报告权限已授予，但已失效（请参阅[修复失效权限](#修复失效权限)）。会显示 **"打开设置"** 按钮。 |
| 🔴 **辅助功能：未启用** | 权限尚未授予。会显示 **"打开设置"** 按钮。 |

## 故障排除

### 修复失效权限

如果您看到橙色状态指示器或"辅助功能：需要刷新"的提示，说明权限已失效。这可能在应用更新或重新构建后发生。

**修复方法：**

1. 打开 **系统设置 > 隐私与安全性 > 辅助功能**。
2. 在列表中找到 **"Window Resize"**。
3. 将其 **关闭**，然后重新 **打开**。
4. 或者，将其从列表中完全移除，然后重新启动应用以重新添加。

### 调整大小失败

如果您看到"调整大小失败"的提示，可能的原因包括：

- 目标应用程序不支持通过辅助功能调整大小。
- 窗口处于 **全屏模式**（请先退出全屏）。
- 辅助功能权限未激活（请在设置中检查状态）。

### 窗口未出现在列表中

调整大小菜单仅显示以下窗口：

- 当前在屏幕上可见的窗口
- 不属于桌面的窗口（例如 Finder 桌面会被排除）
- 不是 Window Resize 应用自身的窗口

如果窗口已最小化到 Dock，则不会出现在列表中。

### 截图不起作用

如果截图未被捕获：

- 请在 **系统设置 > 隐私与安全性 > 屏幕录制** 中授予 **屏幕录制** 权限。
- 确保 **"保存到文件"** 或 **"复制到剪贴板"** 中至少有一个已启用。
