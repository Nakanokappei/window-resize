## 초기 설정

### 손쉬운 사용 권한 부여

Window Resize는 macOS 손쉬운 사용 API를 사용하여 윈도우 크기를 조절합니다. 앱을 처음 실행할 때 권한을 부여해야 합니다.

1. **Window Resize**를 실행합니다. 손쉬운 사용 접근 권한을 요청하는 시스템 대화상자가 나타납니다.
2. **"설정 열기"**를 클릭합니다(또는 직접 **시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용**으로 이동합니다).
3. 목록에서 **"Window Resize"**를 찾아 토글을 켭니다.
4. 앱으로 돌아오면 메뉴 막대에 아이콘이 나타나고 앱을 사용할 준비가 됩니다.

> **참고:** 대화상자가 나타나지 않는 경우 앱의 설정 창에서 직접 손쉬운 사용 설정을 열 수 있습니다([손쉬운 사용 상태](#손쉬운-사용-상태) 참조).

## 윈도우 크기 조절

### 단계별 안내

1. 메뉴 막대에서 **Window Resize 아이콘**을 클릭합니다.
2. **"크기 조절"** 위에 마우스를 올려 윈도우 목록을 엽니다.
3. 현재 열려 있는 모든 윈도우가 **애플리케이션 아이콘**과 함께 **[앱 이름] 윈도우 제목** 형식으로 표시됩니다. 긴 제목은 메뉴의 가독성을 위해 자동으로 줄여집니다.
4. 윈도우 위에 마우스를 올려 사용 가능한 프리셋 크기를 확인합니다.
5. 크기를 클릭하면 즉시 윈도우 크기가 변경됩니다.

### 크기 표시 방식

메뉴의 각 크기 항목은 다음과 같이 표시됩니다:

```
1920 x 1080          Full HD
```

- **왼쪽:** 너비 x 높이 (픽셀 단위)
- **오른쪽:** 라벨 (기기 이름 또는 표준 이름), 회색으로 표시

### 화면을 초과하는 크기

프리셋 크기가 윈도우가 위치한 디스플레이보다 큰 경우, 해당 크기는 **회색으로 표시되어 선택할 수 없습니다**. 이는 윈도우가 화면 경계를 넘어서 크기가 변경되는 것을 방지합니다.

> **멀티 디스플레이:** 앱은 각 윈도우가 어떤 디스플레이에 있는지 감지하고 사용 가능한 크기를 자동으로 조정합니다.

## 설정

메뉴 막대에서 설정을 엽니다: Window Resize 아이콘을 클릭한 다음 **"설정..."**을 선택합니다(단축키: **⌘,**).

### 기본 제공 크기

앱에는 12개의 기본 제공 프리셋 크기가 포함되어 있습니다:

| 크기 | 라벨 |
|------|-------|
| 2560 x 1600 | MacBook Pro 16" |
| 2560 x 1440 | QHD / iMac |
| 1728 x 1117 | MacBook Pro 14" |
| 1512 x 982 | MacBook Air 15" |
| 1470 x 956 | MacBook Air 13" M3 |
| 1440 x 900 | MacBook Air 13" |
| 1920 x 1080 | Full HD |
| 1680 x 1050 | WSXGA+ |
| 1280 x 800 | WXGA |
| 1280 x 720 | HD |
| 1024 x 768 | XGA |
| 800 x 600 | SVGA |

기본 제공 크기는 삭제하거나 편집할 수 없습니다.

### 사용자 지정 크기

목록에 자신만의 크기를 추가할 수 있습니다:

1. **"사용자 지정"** 섹션에서 **너비**와 **높이**를 픽셀 단위로 입력합니다.
2. **"추가"**를 클릭합니다.
3. 새 크기가 사용자 지정 목록에 나타나며 크기 조절 메뉴에서 즉시 사용할 수 있습니다.

사용자 지정 크기를 삭제하려면 옆에 있는 빨간색 **"제거"** 버튼을 클릭합니다.

> 사용자 지정 크기는 크기 조절 메뉴에서 기본 제공 크기 뒤에 표시됩니다.

### 로그인 시 실행

**"로그인 시 실행"**을 활성화하면 macOS에 로그인할 때 Window Resize가 자동으로 시작됩니다.

### 스크린샷

**"크기 조절 후 스크린샷 촬영"**을 활성화하면 크기 조절 후 자동으로 윈도우를 캡처합니다.

활성화하면 다음 옵션을 사용할 수 있습니다:

- **파일로 저장** — 스크린샷을 PNG 파일로 저장합니다. 활성화하면 저장 위치를 선택합니다:
  > **파일명 형식:** `MMddHHmmss_앱이름_윈도우제목.png` (예: `0227193012_Safari_Apple.png`). 기호는 제거되며, 문자, 숫자, 밑줄만 사용됩니다.
  - **데스크탑** — 데스크탑 폴더에 저장합니다.
  - **사진** — 사진 폴더에 저장합니다.
- **클립보드에 복사** — 스크린샷을 클립보드에 복사하여 다른 앱에 붙여넣을 수 있습니다.

두 옵션은 독립적으로 활성화할 수 있습니다. 예를 들어, 파일로 저장하지 않고 클립보드에만 복사할 수 있습니다.

> **참고:** 스크린샷 기능은 **화면 기록** 권한이 필요합니다. 이 기능을 처음 사용할 때 macOS가 **시스템 설정 > 개인정보 보호 및 보안 > 화면 기록**에서 권한을 부여하라는 메시지를 표시합니다.

### 손쉬운 사용 상태

설정 창 하단에 손쉬운 사용 권한의 현재 상태를 나타내는 상태 표시기가 있습니다:

| 표시기 | 의미 |
|--------|------|
| 🟢 **손쉬운 사용: 활성화됨** | 권한이 활성 상태이며 정상적으로 작동 중입니다. |
| 🟠 **손쉬운 사용: 갱신 필요** | 시스템에서 권한이 부여된 것으로 보고하지만 더 이상 유효하지 않습니다([오래된 권한 수정](#오래된-권한-수정) 참조). **"설정 열기"** 버튼이 표시됩니다. |
| 🔴 **손쉬운 사용: 비활성화됨** | 권한이 부여되지 않았습니다. **"설정 열기"** 버튼이 표시됩니다. |

## 문제 해결

### 오래된 권한 수정

주황색 상태 표시기 또는 "손쉬운 사용: 갱신 필요" 메시지가 표시되면 권한이 만료된 것입니다. 이는 앱이 업데이트되거나 다시 빌드된 후에 발생할 수 있습니다.

**수정 방법:**

1. **시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용**을 엽니다.
2. 목록에서 **"Window Resize"**를 찾습니다.
3. 토글을 **끄고** 다시 **켭니다**.
4. 또는 목록에서 완전히 제거한 후 앱을 다시 실행하여 다시 추가합니다.

### 크기 조절 실패

"크기 조절 실패" 경고가 표시되면 가능한 원인은 다음과 같습니다:

- 대상 애플리케이션이 손쉬운 사용 기반 크기 조절을 지원하지 않습니다.
- 윈도우가 **전체 화면 모드**에 있습니다(먼저 전체 화면을 종료하세요).
- 손쉬운 사용 권한이 활성화되지 않았습니다(설정에서 상태를 확인하세요).

### 윈도우가 목록에 나타나지 않음

크기 조절 메뉴에는 다음 조건을 충족하는 윈도우만 표시됩니다:

- 현재 화면에 보이는 상태
- 데스크탑의 일부가 아닌 경우(예: Finder 데스크탑은 제외)
- Window Resize 앱 자체의 윈도우가 아닌 경우

윈도우가 Dock으로 최소화되어 있으면 목록에 나타나지 않습니다.

### 스크린샷이 작동하지 않음

스크린샷이 캡처되지 않는 경우:

- **시스템 설정 > 개인정보 보호 및 보안 > 화면 기록**에서 **화면 기록** 권한을 부여하세요.
- **"파일로 저장"** 또는 **"클립보드에 복사"** 중 하나 이상이 활성화되어 있는지 확인하세요.
